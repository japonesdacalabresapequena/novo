Guia de Tags HTML e CSS

Tags HTML

Aqui estão algumas tags HTML básicas:

  • <p>: Define um parágrafo de texto.
  • <img>: Insere uma imagem na página.
  • <h1>: Define um cabeçalho de nível 1.

Propriedades CSS

Aqui estão três propriedades CSS comuns:

  • Centralização: Para centralizar elementos, você pode usar text-align: center; para textos e margin: 0 auto; para elementos block.
  • Cor: Para definir a cor do texto, use color: nomeDaCor; ou color: #hexadecimal;.
  • Tamanho: Para definir o tamanho de elementos, use propriedades como font-size: para texto e width e height para dimensões de elementos.
